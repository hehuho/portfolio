Définition

Java est un langage de programmation orienté objet. La particularité et l'objectif central de Java est que les logiciels écrits dans ce langage doivent être très facilement portables sur plusieurs systèmes d’exploitation tels que Unix, Windows, Mac OS ou GNU/Linux, avec peu ou pas de modifications. Le langage Java reprend en grande partie la syntaxe du langage C++. Néanmoins, Java a été épuré des concepts les plus subtils du C++. Ses caractéristiques sont que c'est un langage compilé et interprété, sa portabilité, paradigme objet, se veut plus simple que C++, fortement typé, gestion automatique de la mémoire, langage sécurisé, permet l'utilisation de thread. Il y a aussi JEE qui est la version entreprise pour la création d'application web et c'est l'une des raisons de sa popularité en entreprise. Dû à sa popularité et de son coté Open Source, de nombreux frameworks tiers voient le jour dont les plus connus sont Spring, Hibernate, JUnit. Selon divers classements, Java est un des langages les plus populaires et le restera je pense encore un moment
